# AI Marketplace Backend

A production-grade backend built using Spring Boot 3.5, Java 21, Spring AI, PostgreSQL, Redis, Docker, JWT, OAuth2, and Swagger.

## Features

- JWT Authentication
- OAuth2 Login
- Role-Based Access Control
- AI Tool Marketplace
- Categories
- Reviews
- Bookmarks
- Recommendations
- Spring AI + Ollama Chat
- Redis Caching
- Docker Support
- Swagger Documentation

## Tech Stack

- Java 21
- Spring Boot 3.5
- Spring Security
- Spring AI
- PostgreSQL
- Redis
- Docker
- Maven

## Run

```bash
docker compose up -d
```

```bash
mvn spring-boot:run
```

Swagger:

```
http://localhost:8080/swagger-ui.html
```
